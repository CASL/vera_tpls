159: /*E
160:     SNESConvergedReason - reason a SNES method was said to
161:          have converged or diverged

163:    Level: beginner

165:    The two most common reasons for divergence are
166: $   1) an incorrectly coded or computed Jacobian or
167: $   2) failure or lack of convergence in the linear system (in this case we recommend
168: $      testing with -pc_type lu to eliminate the linear solver as the cause of the problem).

170:    Diverged Reasons:
171: .    SNES_DIVERGED_LOCAL_MIN - this can only occur when using the line-search variant of SNES.
172:        The line search wants to minimize Q(alpha) = 1/2 || F(x + alpha s) ||^2_2  this occurs
173:        at Q'(alpha) = s^T F'(x+alpha s)^T F(x+alpha s) = 0. If s is the Newton direction - F'(x)^(-1)F(x) then
174:        you get Q'(alpha) = -F(x)^T F'(x)^(-1)^T F'(x+alpha s)F(x+alpha s); when alpha = 0
175:        Q'(0) = - ||F(x)||^2_2 which is always NEGATIVE if F'(x) is invertible. This means the Newton
176:        direction is a descent direction and the line search should succeed if alpha is small enough.

178:        If F'(x) is NOT invertible AND F'(x)^T F(x) = 0 then Q'(0) = 0 and the Newton direction
179:        is NOT a descent direction so the line search will fail. All one can do at this point
180:        is change the initial guess and try again.

182:        An alternative explanation: Newton's method can be regarded as replacing the function with
183:        its linear approximation and minimizing the 2-norm of that. That is F(x+s) approx F(x) + F'(x)s
184:        so we minimize || F(x) + F'(x) s ||^2_2; do this using Least Squares. If F'(x) is invertible then
185:        s = - F'(x)^(-1)F(x) otherwise F'(x)^T F'(x) s = -F'(x)^T F(x). If F'(x)^T F(x) is NOT zero then there
186:        exists a nontrival (that is F'(x)s != 0) solution to the equation and this direction is
187:        s = - [F'(x)^T F'(x)]^(-1) F'(x)^T F(x) so Q'(0) = - F(x)^T F'(x) [F'(x)^T F'(x)]^(-T) F'(x)^T F(x)
188:        = - (F'(x)^T F(x)) [F'(x)^T F'(x)]^(-T) (F'(x)^T F(x)). Since we are assuming (F'(x)^T F(x)) != 0
189:        and F'(x)^T F'(x) has no negative eigenvalues Q'(0) < 0 so s is a descent direction and the line
190:        search should succeed for small enough alpha.

192:        Note that this RARELY happens in practice. Far more likely the linear system is not being solved
193:        (well enough?) or the Jacobian is wrong.

195:    SNES_DIVERGED_MAX_IT means that the solver reached the maximum number of iterations without satisfying any
196:    convergence criteria. SNES_CONVERGED_ITS means that SNESConvergedSkip() was chosen as the convergence test;
197:    thus the usual convergence criteria have not been checked and may or may not be satisfied.

199:    Developer Notes: this must match finclude/petscsnes.h

201:        The string versions of these are in SNESConvergedReason, if you change any value here you must
202:      also adjust that array.

204:    Each reason has its own manual page.

206: .seealso: SNESSolve(), SNESGetConvergedReason(), KSPConvergedReason, SNESSetConvergenceTest()
207: E*/
